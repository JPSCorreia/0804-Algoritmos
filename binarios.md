### Binário para Decimal

**1111:**
2^0 x 1 = 1 x 1 = 1
2^1 x 1 = 2 x 1 = 2
2^2 x 1 = 4 x 1 = 4
2^3 x 1 = 8 x 1 = 8
= **15**

**101010:**
2^0 x 0 = 1 x 0 = 0
2^1 x 1 = 2 x1 = 2
2^2 x 0 = 4 x 0 = 0
2^3 x 1 = 8 x 1 = 8
2^4 x 0 = 16 x 0 = 0
2^5 x 1 = 32 x 1 = 32
= **42**

**1010:**

### Decimal para Binário

**17:**
17 / 2 = 8 e resto 1
8 / 2 = 4 e resto 0
4 / 2 = 2 e resto 0
2 / 2 = 1 e resto 0
1
= **10001**


**25:**

6 / 2 = 3 e resto 0
3 / 2 = 1 e resto 1
